## Project setup

```
npm install
```

### Run development server

```
npm run start
```

### Run build server

```
npm run start:build
```

### Compile typescript to javascript

```
npm run build
```

### Run tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```

### Image processing endpoint

```
/api/image?filename=fjord&width=400&height=300
```
